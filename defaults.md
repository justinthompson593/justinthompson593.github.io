# EsvApiInterface Defaults

When you install `EsvApiInterface`, run it for the first time, or run `$ EsvApiInterface -def`, you will be prompted to set your defaults. Below is an example of the command line output.

```
______________EsvApiInterface______________

Choose CSS type

1: Default
2: Dark
3: Custom


Enter a number (1-3): 2
Verse Numbers      (0=off / 1=on): 1
Headings           (0=off / 1=on): 1
Subheadings        (0=off / 1=on): 1
Footnotes          (0=off / 1=on): 1
Footnote Links     (0=off / 1=on): 1
Passage References (0=off / 1=on): 1
Words of Christ in red  ( 0 / 1 ): 0

What should happen when you run EsvApiInterface with no input arguments? i.e.
	EsvApiInterface

1: Open and/or save the daily verse
2: Open a random verse
3: Show bookmarks
4: Open and/or save a specific bookmark

Enter a number (1-4): 2
What format should the output be when you run with no input arguments?

1: html
2: text
3: mp3 link

Enter a number (1-3): 1

What should happen when you run EsvApiInterface with one input argument? i.e.
	EsvApiInterface yourArg

1: Assume yourArg is a passage query, open and/or save it
2: Assume yourArg is a preexisting bookmark name, open and/or save the bookmark
3: Assume yourArg is a new passage query to bookmark; you will be prompted for a bookmark name after running

Enter a number (1-3): 1
	1: Open only
	2: Save only
	3: Open and save

	Enter a number (1-3): 1
What format should the output be when you run with one input argument?

1: html
2: text
3: mp3 link

Enter a number (1-3): 1
```
